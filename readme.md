# Apartment

Класс `Apartment` предназначен для представления квартиры с возможностью покупки или аренды.

## Описание

Объект хранит информацию о квартире:
- `price` — цена покупки (в рублях)
- `price_month` — стоимость аренды за месяц (в рублях)
- `address` — адрес квартиры
- `square` — площадь (в м²)
- `available` — доступность (True — доступна, False — сдана)

Все поля защищены через `@property` и проходят валидацию при установке.

---

## Методы

### `rent(months, money)`
Аренда квартиры.

**Параметры:**
- `months` (int) — количество месяцев
- `money` (int) — сумма денег

**Логика:**
- Проверяет корректность типов и значений
- Проверяет доступность квартиры
- Проверяет достаточность средств
- Делает квартиру недоступной (`available = False`)

---

### `buy(money)`
Покупка квартиры.

**Параметры:**
- `money` (int) — сумма денег

**Логика:**
- Проверяет корректность типа и значения
- Проверяет доступность квартиры
- Проверяет достаточность средств

---

## Специальные методы

- `__str__()` — красивый вывод информации о квартире
- `__repr__()` — техническое представление объекта
- `__eq__()` — сравнение двух объектов `Apartment`

---

## Валидация

Каждое поле проходит проверку:
- Цена и аренда — положительные целые числа
- Адрес — непустая строка
- Площадь — положительное целое число
- Доступность — булево значение
